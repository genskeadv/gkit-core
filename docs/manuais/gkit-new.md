# Manual de uso - GKIT New

## Para que serve

O GKIT New organiza o fluxo comercial: prospeccao, contatos, oportunidades,
workflow comercial e tarefas. Ele separa a execucao diaria do cockpit da visao
gerencial do dashboard.

## Quem usa

- Equipe comercial.
- Gestores de novos negocios.
- Usuarios responsaveis por follow-up, propostas e conversao de clientes.

## Como acessar

- Cockpit comercial: `/modulos/gkit-new`
- Clientes: `/modulos/gkit-new/clientes`
- Novo cliente: `/modulos/gkit-new/clientes/novo`
- Contatos: `/modulos/gkit-new/contatos`
- Novo contato: `/modulos/gkit-new/contatos/novo`
- Oportunidades: `/modulos/gkit-new/oportunidades`
- Nova oportunidade: `/modulos/gkit-new/oportunidades/novo`
- Workflow: `/modulos/gkit-new/base/workflow`
- Tarefas: `/modulos/gkit-new/tarefas`
- Dashboard: `/modulos/gkit-new/gestao`

## Visao rapida do fluxo

1. Cadastre o cliente ou lead.
2. Cadastre contatos vinculados.
3. Crie a oportunidade.
4. Use o workflow para gerar tarefas comerciais.
5. Execute follow-ups pelo cockpit.
6. Aprove, rejeite, cancele ou encerre a oportunidade conforme o resultado.

## Telas principais

### Cockpit

E a tela de execucao diaria. Use para realizar tarefas, follow-ups e acoes
comerciais sem sair para listas longas. Formularios operacionais aparecem abaixo
dos cards quando a acao exige preenchimento.

### Dashboard

Use para indicadores e acompanhamento gerencial. Estatisticas e consolidacoes
devem ficar no dashboard, enquanto o cockpit permanece focado em execucao.

### Clientes

Use para registrar empresas, condominios ou pessoas em prospeccao. Ao aprovar
uma oportunidade, o cliente pode seguir para ativacao no fluxo principal.

### Contatos

Use para registrar decisores, influenciadores e pessoas de contato. Mantenha
telefone, e-mail, cargo e vinculo com o cliente.

### Oportunidades

Use para controlar propostas e negociacoes. A oportunidade concentra valor,
etapa, responsavel, previsao, origem e status.

Status comuns:

- Nova.
- Proposta enviada.
- Em negociacao.
- Aprovada.
- Rejeitada.
- Cancelada.
- Encerrada.

### Workflow

Use para cadastrar modelos de tarefas comerciais. O workflow ajuda a padronizar
o que deve acontecer em cada etapa da oportunidade.

### Tarefas

Use para acompanhar tarefas geradas pelo workflow ou criadas manualmente. Ao
concluir uma tarefa, confira se o proximo follow-up ficou coerente.

## Como operar uma oportunidade

1. Confirme se cliente e contatos estao cadastrados.
2. Crie a oportunidade com responsavel, etapa e previsao.
3. Aplique ou acompanhe o workflow correspondente.
4. Execute as tarefas abertas pelo cockpit.
5. Registre cada avancao da negociacao.
6. Ao final, marque como aprovada, rejeitada, cancelada ou encerrada.

## Boas praticas

- Use o cockpit para trabalhar; use o dashboard para medir.
- Nao deixe oportunidade sem responsavel.
- Registre follow-up com proxima acao clara.
- Evite criar contatos duplicados para o mesmo cliente.
- Ao aprovar oportunidade, revise os dados antes de levar o cliente para a base.
